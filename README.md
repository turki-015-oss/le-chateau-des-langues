# Le Château des Langues — v3.0-dev1

إعادة بناء الصفحة الرئيسية لتصبح خريطة مملكة احترافية فقط.

- خريطة كبيرة قابلة للسحب والتكبير.
- تصميم Isometric احترافي.
- الصفحة الرئيسية هي الخريطة مباشرة.
- مبانٍ تفاعلية ودخول مباشر للعوالم المفتوحة.
- مساحات توسع مستقبلية حول المدينة.
- لا جواهر ولا رموز دفع ولا علامة صليب.
- حفظ موقع الكاميرا تلقائيًا.
