# Le Château des Langues — v6.5

## تغييرات القلعة
- تحويل المحكمة الملكية إلى قاعة القواعد وربطها بالمسار `/grammar`.
- تغيير المكتبة الملكية إلى المكتبة.
- تغيير الأكاديمية الملكية إلى قاعة الاختبارات.
- تغيير الاسم العربي من القلعة الملكية إلى القلعة.
- تحديث نص الترحيب إلى Bienvenue / مرحبًا بك / استكشف وتعلّم اللغة الفرنسية.
- تحديث مدخل القلعة إلى عبارة «تعلّم باحترافية».
- دمج المشاهد المعتمدة كخلفيات مع واجهات حقيقية قابلة للنقر، وليست صورًا ثابتة للصفحات.

## قاعة القواعد
- صفحة جديدة متجاوبة وعملية للمستويات A1 وA2 وB1 وB2 وC1 وC2.
- بطاقات مستويات تفاعلية تعرض القسم المختار ومجالاته الأساسية.
- لا تحتوي الصفحة على تمارين أو اختبارات.
- تصميم أخضر داكن وذهبي بطابع سعودي عملي.

## التحقق
- نجح أمر `npm run build` وأضيف المسار `/grammar` إلى البناء النهائي.
