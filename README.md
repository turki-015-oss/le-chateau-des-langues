# Le Château des Langues — v2.0-dev3

دفعة تنفيذية جديدة لعالم Le Café.

## الجديد
- زبون يومي ديناميكي بثلاث شخصيات مختلفة.
- طلب مسموع باللغة الفرنسية لكل زبون.
- مهمة خدمة حقيقية: الاستماع ثم تجهيز الطلب المطابق.
- تحقق فوري من الطلب مع تأثير مباشر على السمعة.
- منع إنهاء العالم قبل تنفيذ طلب الزبون الصحيح.
- تحسين واجهة المهمة للجوال.
