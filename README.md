# Le Château des Langues — v3.0-dev6

## الجديد
- ظهور صورة اللاعب في الشريط العلوي.
- 6 أفاتارات جاهزة للاختيار.
- رفع صورة شخصية من الاستديو وحفظها محليًا.
- تسجيل دخول وإنشاء حساب محلي تجريبي على الجهاز.
- نقل قائمة الخيارات إلى الجهة اليسرى.
- إضافة حفظ التقدم يدويًا.
- إضافة وضع الصفحة العادية بالخانات.
- إضافة زر الرجوع إلى الخريطة التفاعلية.
- إضافة إدارة الملف الشخصي وتسجيل الخروج.

> ملاحظة: نظام الحساب في هذا الإصدار محلي على الجهاز، وليس حسابًا سحابيًا حقيقيًا بعد.
